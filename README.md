# revolut-home-task

> Revolut home task


## Requirements

- [Node.js](https://nodejs.org/en/)
- [Yarn](https://yarnpkg.com/en/docs/install) package manager


## Quick Start

```sh
$ yarn
$ npm run watch
$ open http://0.0.0.0:7000
```

## Production start

```sh
$ yarn
$ npm run build
$ npm start
$ open http://0.0.0.0:7000
```
